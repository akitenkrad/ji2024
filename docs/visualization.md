# Visualization

The Python package `srap-tools` (module `srap_tools`) renders figures from the Rust outputs. Install with `uv sync` at the workspace root; run with `uv run srap-tools <subcommand>`.

## `visualize` — single run

Reads `metrics.csv` (long-format; averaged per round across trials) from `--results_dir` (default `results/latest`).

```bash
uv run srap-tools visualize
uv run srap-tools visualize --results_dir results/20260525_120000
```

Outputs to `{results_dir}/figures/`:

- `welfare_timeseries.png` — SW and Avg r_size over rounds (satisfaction).
- `fairness_timeseries.png` — Gini and F(V,NV) over rounds (fairness).
- `final_metrics.png` — the final-round metric summary (bar chart).

## `visualize-sweep` — sweep / POA

Reads `sweep_summary.csv` and/or `poa_history.csv` from `--sweep_dir` (default `results/latest`).

```bash
uv run srap-tools visualize-sweep
uv run srap-tools visualize-sweep --sweep_dir results/20260525_120000_sweep
```

Outputs (depending on which CSV is present):

- `sweep_sw_by_subset.png` — average SW by resource subset, coloured by entry condition (the r_size > r_random ordering is clearest with matched seeds — see [Reproduction](reproduction.md)).
- `poa_convergence.png` — POA best/mean fitness over generations (Fig. 4-style; best is monotonically non-decreasing under elitism).

## `show-experiment-settings`

Prints the run/sweep/poa config and LLM metadata for a results directory (`--results-dir`, `--json` for JSON).
