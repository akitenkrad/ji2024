# 可視化

Python パッケージ `srap-tools` (モジュール `srap_tools`) が Rust の出力から図を生成する．workspace ルートで `uv sync` してインストールし，`uv run srap-tools <subcommand>` で実行する．

## `visualize` — 単一実行

`--results_dir` (既定 `results/latest`) の `metrics.csv` (long-format; ラウンドごとに試行平均) を読む．

```bash
uv run srap-tools visualize
uv run srap-tools visualize --results_dir results/20260525_120000
```

出力 `{results_dir}/figures/`:

- `welfare_timeseries.png` — SW と Avg r_size のラウンド推移 (満足度)．
- `fairness_timeseries.png` — Gini と F(V,NV) のラウンド推移 (公平性)．
- `final_metrics.png` — 最終ラウンドの指標サマリ (棒グラフ)．

## `visualize-sweep` — sweep / POA

`--sweep_dir` (既定 `results/latest`) の `sweep_summary.csv` および/または `poa_history.csv` を読む．

```bash
uv run srap-tools visualize-sweep
uv run srap-tools visualize-sweep --sweep_dir results/20260525_120000_sweep
```

出力 (存在する CSV に応じて):

- `sweep_sw_by_subset.png` — 資源サブセット別の平均 SW を入室条件で色分け (r_size > r_random の順序は seed を揃えた比較で最も明瞭; [再現](reproduction.ja.md) 参照)．
- `poa_convergence.png` — POA の最良/平均適応度の世代推移 (Fig.4 風; エリート保存で最良は単調非減少)．
